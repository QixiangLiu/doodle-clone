Doodle-Clone Documentation

Overview:

Our doodle clone utilized mainly JavaScript, HTML, and CSS. To perform data storage, we used localStorage, a built in HTML function that takes in a key which stores any sort of information you want it to. We utilized this store the object-converted string of event objects.
